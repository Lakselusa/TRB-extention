Why this exists

Combine multiple IFCs without leaving the browser.

The Trimble Connect Desktop app can export a combined .trb from multiple loaded models. This extension brings that workflow to the web — same idea, no install.

Lives in the viewer

Installs as a native 3D Viewer extension. Appears in the sidebar alongside layers, todos, and clashes — no context switching.

Reads the current view

Uses the Workspace API to access models already loaded and converted to TrimBIM by Trimble Connect's own pipeline.

One-click export

Pick the models you want to combine, name your file, and download. The browser handles the rest.

Get started

Install in three steps.

01

Open Project Settings

In your Trimble Connect project, navigate to the Project Settings page from the left navigation panel.

02

Add a custom extension

Go to the Extensions tab and click Add custom extension. Paste the manifest URL on the right into the field.

03

Open the 3D Viewer

Load some IFC models, then open the TrimBIM Export extension from the viewer sidebar. Select your models and click Export.

Manifest URL

Paste this into Trimble Connect

https://trimbim-tools.xyz/extension-manifest.json

Requires admin permission on the project. The extension runs entirely in your browser — your model data never leaves the Trimble Connect environment.
